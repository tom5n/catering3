# Prostě Petr - Catering

Webová stránka pro catering služby vytvořená v Next.js.

## Instalace

```bash
npm install
```

## Spuštění vývojového serveru

```bash
npm run dev
```

Otevřete [http://localhost:3000](http://localhost:3000) ve vašem prohlížeči.

## Video pro Hero sekci

Pro správné zobrazení hero sekce vložte video soubor `hero-video.mp4` do složky `public/`.

## Struktura projektu

- `app/` - Hlavní aplikace Next.js
  - `components/` - React komponenty
    - `Header.tsx` - Header s transparentním efektem při scrollování
    - `Hero.tsx` - Hero sekce s videem na pozadí
  - `globals.css` - Globální styly s fonty Montserrat a Playfair Display
  - `layout.tsx` - Root layout
  - `page.tsx` - Hlavní stránka

## Barvy

## Fonty

- **Montserrat** - pro hlavní text
